Instructions for Pacing
  1. Place pacing ECG electrodes AND pacer pads on chest per package instructions
  2. Turn monitor/defibrillator ON, set to PACER mode
  3. Set PACER RATE (ppm) to desired rate/min. (Can be adjusted up or down based on clinical response once pacing is established)
  4. Increase the milliamperes (mA) of PACER OUTPUT until electrical capture (pacer spikes aligned with QRS complex; threshold normally 65‚Äê100mA)
  5. Set final mA to 10mA above this level
  6. Confirm pulse is present
  7. Must change pacing pads hourly to avoid burns

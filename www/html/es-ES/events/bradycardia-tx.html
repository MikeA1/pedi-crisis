Treatment
Pick One:
  • Start chest compressions, go to CARDIAC ARREST
    • Epinephrine 10 MICROgrams/kg IV
    • Epinephrine MICROgrams (10 MICROgrams/kg) IV
    • Confirm NSR. If heart block or slow junction/ventricular, call Intraoperative Electrophysiology
    • Call for transcutaneous pacer (go to INSTRUCTIONS FOR PACING and start when available)
    • Consider ECMO
  • Treat underlying etiology:
    • If hypoxia (common): Increase O2 to 100%, ensure ventilation, and go to HYPOXIA
    • If vagal: Atropine 0.01-0.02 mg/kg IV
    • If vagal: Atropine mg (0.01-0.02 mg/kg) IV
    • If Surgical Stimulation: Stop stimulation and if laparoscopy, desufflate
    • If Ca-Channel Blocker Overdose: Calcium chloride 10-20 mg/kg IV or Calcium gluconate 50 mg/kg IV. If ineffective, Glucagon as dosed below
    • If Ca-Channel Blocker Overdose: Calcium chloride mg IV (10-20 mg/kg) IV or Calcium gluconate mg (50 mg/kg). If ineffective, Glucagon as dosed below
    • If Beta-Blocker Overdose: Glucagon 50 MICROgrams/kg IV, then 0.07 mg/kg/hour IV infusion (MAX 5 mg/hr) and check blood sugar
    • If Beta-Blocker Overdose: Glucagon MICROgrams (50 MICROgrams/kg) IV, then 0.07 mg/kg/hour IV infusion (MAX 5 mg/hr) and check blood sugar

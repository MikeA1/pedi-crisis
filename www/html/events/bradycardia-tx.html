Treatment
  • If hypotensive, pulseless, or poor perfusion, start chest compressions, go to CARDIAC ARREST
    • EPINEPHrine 10 MICROgrams/kg IV
    • EPINEPHrine MICROgrams (10 MICROgrams/kg) IV
    • Call for transcutaneous pacer (go to INSTRUCTIONS FOR PACING and start when available)
  • If NOT hypotensive or pulseless, treat underlying etiology:
    • If hypoxia (common): Give 100% O2, ensure ventilation, and go to HYPOXIA
    • If vagal: Atropine 0.01-0.02 mg/kg IV
    • If vagal: Atropine mg (0.01-0.02 mg/kg) IV
    • If Surgical Stimulation: Stop simulation and If laparoscopy, desufflate
    • If Ca-Channel Blocker Overdose: Calcium chloride 10-20 mg/kg IV or Calcium gluconate 50 mg/kg. If ineffective, Glucagon as dosed below
    • If Ca-Channel Blocker Overdose: Calcium chloride mg (10-20 mg/kg) IV or Calcium gluconate mg (50 mg/kg). If ineffective, Glucagon as dosed below
    • If Beta-Blocker Overdose: Glucagon 50 MICROgrams/kg IV, then 0.07 mg/kg/hour IV infusion (MAX 5 mg/hr) and check blood sugar
    • If Beta-Blocker Overdose: Glucagon MICROgrams (50 MICROgrams/kg) IV, then 0.07 mg/kg/hour IV infusion (MAX 5 mg/hr) and check blood sugar

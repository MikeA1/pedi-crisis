Crisis Management (if Severe)
  • Notify surgeon, call for help and code cart
  • Call for echocardiography
  • Check pulse. If no pulse or HR<30:
    • Start chest compressions
    • Give EPINEPHrine 1-10 MICROgrams/kg IV/IO
    • Give EPINEPHrine MICROgrams (1-10 MICROgrams/kg) IV/IO
    • If cardiac arrest, go to CARDIAC ARREST
    • Consider ECMO

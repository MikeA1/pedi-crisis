Visible Scrollbar
When on, the scrollbar is always visible.
When off, the scrollbar is only visible while scrolling.
Show Event Diagnosis
When on, the event list displays select diagnoses.
Navigation Gestures
When on, navigate between sections by swiping from left-to-right or right-to-left. For example, move from Phone Numbers to Events by swiping left-to-right.
Accessibility Mode
When on, text is larger and darker.
